About addons

Addons are designed to let you add more functionality to RPi-Monitor.

The realease of RPi-Monitor provides the following addons:
  • about - this addons
  • custom - custom addons using <iframe> to display external html page
  • example - example of addons showing how to develop addons fully integrated into RPi-Monitor
  • shellinabox - Allow ssh access to your server from your browser
  • top3 - Show the top 3 processes using cpu and rpimonitord cpu usage

Addons are configured into /etc/rpimonitor/data.conf. You can customize RPi-Monitor configuration to add or remove addons.

Refer to manage and/or comments in configuration file to see in detail how to use addons.

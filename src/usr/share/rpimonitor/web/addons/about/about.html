About addons

Addons are designed to let you add more functionality to RPi-Monitor.

This release of RPi-Monitor provides the following addons:
  • about - This addon
  • example - Example of addons showing how to develop addons that are fully integrated with RPi-Monitor
  • custom - Custom addon using <iframe> to display external html page. This addon is use for shellinabox or Hawkeye webcam - See documentation for details.
  • top3 - Show the top 3 processes using cpu and rpimonitord cpu usage

Addons are configured through /etc/rpimonitor/data.conf. You can customize the RPi-Monitor configuration to add or remove addons.

Refer to comments in configuration file to see in detail how to manage and use addons.
